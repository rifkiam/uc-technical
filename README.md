# United Creative Technical Test

This repository is intended to act as submission medium for United Creative technical test.

## Contents

`/backend` uses Golang
`/frontend` uses Next.js

## How to run service
### Using script

On Linux, run:
`./run.sh`

On Windows, run:
`.\run.ps1`

### Individually

`/backend`:
1. `go mod tidy`
2. `go run .`

`/frontend`:
1. `npm install`
2. `npm run dev`

or if using yarn:
1. `yarn`
2. `yarn dev`
